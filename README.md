
# 🧠 Portfolio Developer - v0.1

A Flask-based web app that helps users generate professional-looking portfolio pages by entering structured prompts. It supports dynamic template switching, intuitive UI, and detailed parsing of user data.

---

## ✨ Features

- ✅ Prompt-based portfolio creation
- 🎨 Random CSS themes on refresh
- 🧩 Well-structured user data parsing
- 📱 Responsive, interactive landing UI
- 🔎 Built-in help and example prompt loader
